Вивчаємо CSS властивості оформлення блоків та властивість background

Властивість `border` — встановлює товщину, стиль та колір межі (кордону) елемента. Не успадковується.

css

анатомія властивості

border: товщина стиль колір;

Приклад запису

border: 2px solid #green;

Основні стилі:

  • - solid — суцільна лінія;
  • - dotted — лінія крапками;
  • - dashed — пунктирна лінія;
  • - double — подвійна лінія.

Приклад запису border для окремих сторін блоку:

  • border-top: 6px solid red;
  • border-right: 12px dotted blue;
  • border-bottom: 8px double green;
  • border-left: 10px dashed yellow;

Скасування border для однієї сторони border-top: none;

Властивість `outline` — встановлює товщину, стиль та колір обвідки навколо елемента. Не успадковується.

outline знаходиться навколо елементу (обводить елемент ззовні), в той час, як border знаходиться всередині елементу та окреслює його кордони.

border: 6px solid red;

outline: 12px solid yellowgreen;

border-radius (значення може бути в пікселях, відсотках і т.п.)

Встановлює радіус заокруглення кутів блоку.

Властивість не успадковується.

Приклад запису для усіх сторін - border-radius: 50px;

Разом із цім заокруглення впливає і на border і на outline, якщо вони встановлені для елемента.

приклад як зробити коло

.circle {
border-radius: 50%;
width: 300px;
height: 300px;
background-color: #794f45;
}

  • box-shadow
  • Додає тінь до елементу
  • Властивість не успадковується
  • box-shadow: зсув_по_гориз зсув_по_верт радіус(розмиття) розмір колір
  • inset - тінь в середені об'єкту

Властивість opacity — встановлює рівень прозорості елемента. Не успадковується.

⬇⬇⬇прозорий елемент⬇⬇⬇ ⬇⬇⬇НЕпрозорий елемент⬇⬇⬇ ⬇⬇⬇прозорість елементу 50%⬇⬇⬇
0 — повна прозорість; 1 — повна непрозорість; 0.5 — прозорість 50%

Властивість visibility — управляє відображенням елемента.

hidden — ховає об'єкт (з об'єктом не можна взаємодіяти).

visible — відображає об'єкт (за замовченням);

Властивість background

Дозволяє одночасно задати кілька параметрів фону:

  • визначити колір
  • встановити фонове зображення
  • задати початкове положення фонового зображення
  • визначити як повторюватиметься фонове зображення
  • встановити, чи прокручуватиметься фонове зображення разом із вмістом
  • масштабувати фонове зображення відповідно до заданих розмірів
  • вказати градієнтну заливку фону

Встановлюємо градієнтну заливку елемента:

background: linear-gradient( 16deg, rgba(63, 94, 251, 1) 0%, rgba(252, 70, 107, 1) 100% );

Повторення фонового зображення:

background-repeat: repeat; — за замовченням (зображення повторюється для заповнення блоку)
не повторювати (зображення не повторюється для заповнення блоку)
повторювати по горизонталі (зображення повторюється лише по горизонталі для заповнення блоку)
повторювати по вертикалі (зображення повторюється лише по вертикалі для заповнення блоку)

Задаємо початкове положення фонового зображення:

background-position: 0 0; перше значення — розташування по горизонталі (значення може бути у відсотках, px, словом), друге значення — по вертикалі.

Якщо задано значення лише для горизонталі чи лише по вертикалі то друге значення за замочуванням буде center

0 0 (або left top) — за замовченням (зображення буде починатись з верхнього лівого кута блоку)
center (або 50%) — центрує зображення посередині блоку
top
left
bottom
right

Чи прокручуватиметься фонове зображення разом із вмістом елемента

scroll — скролиться разом з блоком (за замовченням)

fixed — фонове зображення НЕ скролиться разом з блоком (позиціонується відносно в'юпорту, а не відносно блоку)

Масштабуємо фонове зображення згідно з заданими розмірами у цифрах

Перше значення — ширина, друге — висота (значення можуть бути у відсотках, px). Або спеціальне значення.

background-size: auto auto; — за замовчуванням.
Якщо хочемо працювати від висоти, то першим обовʼязково ставимо auto: background-size: auto 340px;
Можемо написати тільки ширину, і висота підтягнеться автоматично та пропорційно: background-size: 290px;
Можна писати розмір у відсотках: background-size: 70%;

Є спеціальні значення:

- contain — масштабує зображення таким чином, щоб картинка повністю помістилася всередину блоку. Тобто по найменшій частині картинки (висота, якщо зображення горизонтальне, або ширина). Підходить для фотографій та карток товарів.

- cover — масштабує зображення так, щоб воно зайняло увесь блок (тому частину ми не будемо бачити). Тобто по найбільшої частини картинки. **Підходить для фонових зображень які не несуть в собі конкретної інформації, а використовуються для декору та підкреслення основної інформації (контенту).

Універсальна властивість (універсальний запис — скорочений запис властивості) в більшості випадків використовується для додавання фонового зображення, позиціювання його, розмір зображення та чи повторюється зображення.

Приклад запису:
background: green url("../img/lesson/bg-2.jpg") 0 0 / cover no-repeat;

Також можливо встановлення кількох фонових зображень та градієнтів для блоку.

Наприклад:

background: url("../img/lesson/image-2.png") 0 50px repeat-x,
radial-gradient( 837.78% 378.29% at 69.44% -79.86%, rgba(31, 35, 39, 0) 0%, #191c1f 51.04%, rgba(31, 35, 39, 0) 100% ),
#191c1f url("../img/lesson/bg-4.jpg") center / cover no-repeat;

У цьому прикладі використані 2 різних картинки для background, та додатково накладений затемнюючий градієнт для кращої читальності світлого тексту на тлі світлої картинки. Також на випадок, якщо зображення з якоїсь причини не відобразиться, встановлений темний колір фону (#191c1f)


Оформлення вітальні. Які м'які меблі краще вибрати?

Вітальня – це основна кімната у кожному будинку. Саме вона служить для прийому гостей, проведення часу всією сім'єю та просто для відпочинку у приємній теплій атмосфері. Оформляючи свій будинок та вітальню зокрема, всі намагаються зробити її максимально зручною, комфортною та красивою. Основним та центральним елементом оформлення вітальні, як і будь-якої іншої кімнати, є м'які меблі.

Будь-які меблі для вітальні повинні бути не тільки стильними і красивими, але і функціональними. На таких меблів має бути зручно та приємно відпочивати та приймати гостей. Однак сьогодні існує величезна різноманітність найрізноманітніших м'яких меблів для вітальні, яка підходить для кімнат різних розмірів і форм. На чому варто зупинити свій вибір? Як краще оформити свою вітальню, щоб вам було комфортно, а гості хотіли приходити до вас знову та знову?

На сайті інтернет магазину Кровато представлено велику різноманітність м'яких меблів для вітальні, серед яких кожен зможе вибрати для себе найбільш вдалий варіант. У нас ви знайдете:

  • Прямі дивани. Такі дивани вважаються класикою. Вони підходять для віталень будь-яких форм та розмірів. Прямий диван виглядає стильно та лаконічно, на ньому зручно відпочивати, приймати гостей або навіть спати. Прямі дивани можуть стояти біля стіни, так і в центрі кімнати. Багато моделей прямих диванів легко розкладаються, перетворюючись на повноцінне спальне місце. Також більшість диванів мають великі та місткі ніші, які є додатковим місцем для зберігання;
  • Кутові дивани Кутові дивани вважаються найзручнішими та комфортнішими. Вони ідеально підійдуть для приємного проведення часу в колі сім'ї або друзів. Кутові дивани в основному використовуються для оформлення великих віталень, так як вони займають багато місця, і для кімнат з невеликою площею просто не підійдуть;
  • Тахти. Тахта - це одна з варіацій дивана, але стильніша і мінімалістична. Виглядають тахти дуже незвично і можуть підійти для будь-якого інтер'єру. Також тахти більше ніж звичайні дивани підходять для сну, оскільки в їх основі не пружинні блоки, а дерев'яні ламелі або ортопедичні матраци;
  • Крісла. Крісло – це особливий предмет інтер'єру. Сучасні дизайнери не часто використовують його, вважаючи застарілим, проте саме крісла створюють особливий затишок та комфорт у будь-якій кімнаті. Сьогодні існує величезна різноманітність класичних та сучасних моделей крісел, що дозволяє кожному відшукати ідеальне крісло для свого будинку;
  • Набір меблів. Комплект м'яких меблів – це відмінне рішення для тих, хто хоче меблювати свою вітальню не лише диваном, а й кріслами. Існують набори з одним або двома кріслами, із прямими або кутовими диванами. Основна перевага будь-якого готового меблевого гарнітура – це те, що предмети меблів у ньому ідеально поєднуються один з одним, і вам не потрібно буде витрачати свій час та сили на їхній підбір.

Підбір м'яких меблів для вітальні – це заняття непросте. Однак при виборі та покупці меблів для своєї вітальні пам'ятайте, що якісні м'які меблі повинні бути не тільки красивими, але й зручними для вас і всіх мешканців вашого будинку.
